# Flutter AI 相机 — 界面设计

## 一、整体导航结构

```
App
├── 取景器（主界面，默认进入）
├── 成片预览（仅从相册点击进入）
├── 相册（底部右下角入口）
└── 设置（齿轮图标，LUT 管理、构图辅助开关等）
```

---

## 二、取景器主界面（核心）

### 布局结构（竖屏）

```
┌─────────────────────────────────┐
│ [设置⚙️]              [闪光灯⚡] │ ← 顶部栏（半透明渐变）
│                                 │
│                                 │
│                                 │
│         CameraPreview           │ ← 全屏预览区域
│      + CompositionOverlay       │   （构图网格、人脸框、水平仪）
│      + ShaderMask(LUT 预览)     │
│                                 │
│                                 │
│                                 │
│  [构图💡]    [快门⚪]    [相册🖼] │ ← 底部控制栏
│           [切换📷]              │
└─────────────────────────────────┘
```

### 顶部栏元素

- 左侧：设置图标（进入设置页）
- 右侧：闪光灯状态（自动/开/关/常亮），点击循环切换
- 可选：AI 构图状态指示器（小图标，亮起表示检测到主体）

### 底部控制栏

- **左侧**：构图辅助开关（💡 图标，点击切换网格/提示/关闭）
- **中央**：大圆形快门按钮（按下有缩放反馈，长按可连拍）
- **快门下方**：前后摄切换图标
- **右侧**：相册缩略图（圆角方形，默认显示占位图标；拍照后自动更新为最新照片缩略图，点击进入相册）

### LUT 预设选择器（底部控制栏上方，可滑动）

```
┌─────────────────────────────────┐
│  ◀ [原片] [青橙] [胶片] [高调] ▶ │ ← 横向滚动预设卡片
└─────────────────────────────────┘
```

- 每个预设显示缩略图 + 名称
- 点击即时切换预览效果（GPU Shader 实时渲染）
- 默认隐藏，上滑或点击预设区域展开

## 三、构图辅助 Overlay

### 绘制层级（从底到顶）

1. `CameraPreview`（相机原始画面）
2. `ShaderMask`（LUT 色彩叠加）
3. `CustomPainter`（构图辅助层）：
   - 三分线 / 黄金螺旋（半透明白线，可切换）
   - 人脸检测框（绿色圆角矩形，带置信度阈值过滤）
   - 水平仪（底部中央，倾斜时显示箭头/角度）
   - 轻提示文案（如"主体偏左"、"注意水平"，2 秒自动消失）

### 交互

- 点击 💡 图标循环：关闭 → 仅网格 → 网格+AI 提示 → 关闭
- AI 提示默认轻量，不遮挡画面主体
- 可设置中关闭"构图评分"功能

## 四、成片预览界面

### 触发

仅从相册点击某张照片进入（拍照后不自动弹出，保持取景器界面连续操作）

### 拍照后行为

- 拍照完成后，照片自动保存到系统相册
- 左下角相册按钮的缩略图立即更新为最新拍摄的照片
- 取景器保持显示，不中断拍摄流程
- 用户可随时点击相册按钮查看历史照片

```
┌─────────────────────────────────┐
│ [← 返回]              [分享📤]  │ ← 顶部栏
│                                 │
│                                 │
│         处理后的成片             │ ← 全图显示（可双指缩放）
│                                 │
│                                 │
│                                 │
│  [重新编辑✏️]     [删除🗑️]      │ ← 底部操作栏
└─────────────────────────────────┘
```

### 编辑模式（点击"重新编辑"进入）

- 底部弹出 LUT 预设选择器（同取景器）
- 可选参数滑块：曝光、对比度、饱和度、暗角
- 实时预览，点击"应用"保存到相册

## 八、动效与微交互

| 场景 | 动效 |
|------|------|
| 快门按下 | 按钮缩小 10% + 屏幕闪白 100ms |
| LUT 切换 | 0.3s 淡入淡出过渡 |
| 构图提示出现 | 从底部滑入，2s 后淡出 |
| 人脸框检测成功 | 绿色框 0.2s 缩放弹跳 |
| 水平仪校准 | 箭头归中时微震动（haptic feedback） |
| 前后摄切换 | 旋转 180° 过渡动画 |

## 九、颜色与字体规范

### 主题：深色模式为主（取景器场景）

| 元素 | 颜色 |
|------|------|
| 背景 | `#000000`（纯黑，OLED 省电） |
| 控制栏 | `rgba(0,0,0,0.6)` 半透明 |
| 图标/文字 | `#FFFFFF`（白色） |
| 构图网格 | `rgba(255,255,255,0.4)` |
| 人脸框 | `#4CAF50`（绿色） |
| 警告提示 | `#FF9800`（橙色） |
| 快门按钮 | 白色外圈 + 透明内圈 |

### 字体

系统默认（iOS: SF Pro, Android: Roboto）
- 提示文案：14sp，中等字重
- 预设名称：12sp，常规字重
